# Sudoku3D
Sudoku3D bringt das bekannte Logikspiel auf ein völlig neues Level. Wechsle zwischen mehreren Ebenen, entdecke Zusammenhänge im dreidimensionalen Spielfeld und stelle dein logisches Denken auf die Probe. Mit verschiedenen Schwierigkeitsgraden und optionalem Noob-Modus ist Sudoku3D perfekt für Anfänger und Profis.
